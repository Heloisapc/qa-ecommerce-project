Título: Sistema permite login com credenciais inválidas

Descrição:
Ao inserir usuário e senha incorretos, o sistema permite o acesso ao sistema, o que não é o comportamento esperado.

Passos para reproduzir:
1. Acessar o site https://www.saucedemo.com/
2. Informar usuário válido
3. Informar senha inválida
4. Clicar no botão de login

Resultado esperado:
Sistema deve exibir mensagem de erro e não permitir acesso

Resultado obtido:
Sistema permite o acesso ao sistema mesmo com credenciais inválidas
